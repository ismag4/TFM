# TFM
Warm natural inflation

GQ_smooth.dat: archivo necesario en todos los códigos para obtener el growing mode (G(Q))

Modelos:
  - Códigos python con los modelos Ups ~ T^3 (wi_natural) y Ups modificado (wi_natural_modificada)

Cobaya:
 - CMB_restrictions:
    -  Códigos para inferir parámetros a partir de los observables As, ns y r. Tanto para Ups ~ T^3 y Ups modificado en sus respectivas carpetas
 - likelihood_Planck+BICEP:
    - Código para inferir parámetros a partir de las funciones de correlación de Planck y BICEP/Keck para el modelo Ups ~ T^3
